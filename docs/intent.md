## Reformulated understanding
- The product is a **mobile-friendly platform** for the public.
- It has two core functions:
  - **Annual events indicator**: an easy way to see yearly events (calendar + highlights).
  - **Tourist guide**: clear, basic information to help visitors.
- Content policy:
  - Provide **basic, clear, relevant** information.
  - **No hidden/undisclosed information**.
- Offline:
  - **Offline support is important** (at least for previously loaded content).

## Users
- Public / anyone who wants event and tourism information on mobile.

## Boundaries (out of scope for now)
- No private/hidden information features.
