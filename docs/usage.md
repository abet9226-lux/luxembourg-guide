# Usage guide

## Events

- Use **Search** to filter by title/location/date text.
- Use **Month** and **Area** filters to narrow results.
- Use **Year view** to see events grouped by month.

## Guide (Places)

- Tap a category to browse places.
- Use **Place area** filter to show only a neighborhood/area.

## Saved (Offline)

- Tap an item and choose **Save for offline**.
- Use **Saved area** filter to find saved items by neighborhood/area.

### Backup / Restore

- **Backup** downloads a JSON file with your saved + custom items.
- **Restore** will:
  - download a safety backup first
  - then let you pick a backup JSON to restore

## Maps

- In any detail page, click **Show map preview** (loads Google Maps).
- If offline, the preview won’t load, but **Open in Maps** still works once you’re online.

## Install (PWA)

- If available, click the **Install** button in the top bar.
- Otherwise use your browser menu and choose **Install** / **Add to Home Screen**.
